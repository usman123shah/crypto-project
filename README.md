# 🚀 Crypto Prediction System

A Python-based cryptocurrency prediction system that analyzes market data and news to predict price trends.

---

## 📌 Features

- 📊 Real-time crypto data fetching
- 🧠 Price prediction system
- 📰 News-based analysis
- 📈 Accuracy measurement
- 🖥️ Interactive UI using Streamlit

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas, NumPy
- APIs (for crypto data & news)

---

## 📂 Project Structure

```
crypto-project/
│── app.py                # Main application
│── predictor.py         # Prediction logic
│── data_manager.py      # Data handling
│── news_manager.py      # News analysis
│── measure_accuracy.py  # Accuracy evaluation
│── requirements.txt     # Dependencies
```

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 🔐 Environment Variables

Create a `.env` file and add your API keys:

```
API_KEY=your_api_key_here
```
